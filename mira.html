Robotic Arm Learning Platform (WPI Capstone Project)

Objective:

Design and develop an educational platform to facilitate learning about robotic arm kinematics, control systems, and programming.

My Role:

Lead Engineer (Hardware & Software)

Project Highlights:

  • Modular Control Board Design: Designed interconnected modular control boards for each robotic arm link.
  • CAN Bus Communication: Implemented a CAN bus communication protocol to enable communication between the central computer and individual link controllers.
  • 3D Printed Parts: Designed all robotic arm components for fabrication within the WPI 3D printer lab, ensuring accessibility and cost-effectiveness.
  • Real-Time Interactive GUI: Developed a user-friendly graphical user interface (GUI) for real-time control of each robotic arm joint.
  • Automatic Kinematics Calculations: Implemented software routines for automatic forward and inverse kinematic calculations, allowing users to visualize and control the arm's movement based on desired joint positions.
  • Live Demonstration: Successfully presented the complete robotic arm learning platform with a live demonstration to peers and professors at WPI.

Technical Skills Developed:

  • Hardware Design
  • Embedded Systems Programming
  • CAN Bus Communication Protocols
  • User Interface Design
  • Kinematics (Forward & Inverse)
  • Real-Time Control Systems

Project Impact:

This project provided a valuable educational tool for students to explore robotic arm principles. The modular design allows for future expansion with new link configurations, promoting continued learning and exploration.

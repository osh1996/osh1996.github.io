Northeastern Graduate School: GPS Project (Mobile Robotics)

Project Focus: Sensor Fusion for Enhanced Navigation

Teamwork and Learning Objectives:

  • Collaborated with a team to evaluate various mobile robot navigation sensors (pros & cons).
  • Gained expertise in writing low-level driver code for embedded sensors.
  • Integrated sensor data with Robot Operating System (ROS) for streamlined robotics control.
  • Explored sensor fusion techniques to improve robot perception and navigation accuracy.

Project Activities:

  • Integrated diverse sensor types: satellite communication devices, inertial measurement units (IMUs), and additional sensors.
  • Combined strengths of three sensor types to achieve the most accurate GPS location estimation.
  • Conducted real-world data collection by attaching sensors to a car and recording sensor fusion data streams.

Project Outcome:

Successfully generated a GPS map of the car's driven route with an accuracy of within a few feet, demonstrating the effectiveness of sensor fusion.

Key Skills Developed:

  • Sensor Evaluation and Selection
  • Low-Level Driver Code Development
  • ROS Integration
  • Sensor Fusion Techniques
  • Real-World Data Collection and Analysis

This project showcases the ability to work effectively in a team, apply theoretical knowledge to real-world scenarios, and contribute to the development of robust navigation systems for mobile robots.
